# Retail Profit Optimization & Loss Analysis Dashboard (Excel)

## Project Overview
This Excel dashboard analyzes sales, profit, and discount impact to identify loss-making products and optimize profitability.

## Key Metrics
- Total Sales
- Total Profit
- Profit Margin
- Average Discount

## Insights
• High discounts significantly reduce profit  
• Tables and Bookcases are loss-making subcategories  
• West region generates highest profit  

## Tools Used
Excel, Pivot Tables, Data Visualization
